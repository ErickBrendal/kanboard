📊 Medidas DAX Recomendadas

Adicione estas medidas no modelo de dados do Power BI:

// Total de Demandas
Total = COUNTROWS(Demandas)

// Taxa de Conclusão
Taxa_Conclusao = DIVIDE(CALCULATE(COUNTROWS(Demandas), Demandas[Status]="Concluída"), [Total], 0)

// Demandas em Atraso
Em_Atraso = CALCULATE(COUNTROWS(Demandas), Demandas[Em_Atraso]="Sim")

// Alta Prioridade
Alta_Prioridade = CALCULATE(COUNTROWS(Demandas), Demandas[Prioridade_Texto]="Alta")

// Índice de Saúde do Projeto (0-100)
Saude = MAX(0, 100 - [Em_Atraso] * 10 - [Alta_Prioridade] * 2)
