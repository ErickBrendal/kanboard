Dataset Publicado no Power BI Service

O dataset EBL Fast Track Salesforce está publicado no workspace da conta admebl@eblsolucoescorporativas.com com 4 tabelas: Demandas (105 linhas), Fases, KPIs e Responsáveis.

2

Conexão via Power Query M (Power BI Desktop)

Cole este código no Editor Avançado do Power Query para conectar diretamente ao Kanboard:

let
  url = "http://kanboard.eblsolucoescorp.tec.br/jsonrpc.php",
  body = "{""jsonrpc"":""2.0"",""method"":""getAllTasks"",""id"":1,""params"":{""project_id"":11,""status_id"":1}}",
  headers = [#"Content-Type"="application/json", Authorization="Basic " & Binary.ToText(Text.ToBinary("admin:Senha@2026"), BinaryEncoding.Base64)],
  response = Web.Contents(url, [Headers=headers, Content=Text.ToBinary(body)]),
  json = Json.Document(response),
  result = json[result],
  table = Table.FromList(result, Splitter.SplitByNothing(), null, null, ExtraValues.Error)
in
  table
3

Medidas DAX Prontas

Cole estas medidas na visualização de dados do Power BI Desktop:

// Total de Demandas
Total Demandas = COUNTROWS(Demandas)

// Taxa de Conclusão
Taxa Conclusão = DIVIDE(CALCULATE(COUNTROWS(Demandas), Demandas[status]="Implementado"), [Total Demandas], 0)

// Demandas Alta Prioridade
Alta Prioridade = CALCULATE(COUNTROWS(Demandas), Demandas[pri]="Alta")

// Demandas em Andamento
Em Andamento = CALCULATE(COUNTROWS(Demandas), Demandas[fase] IN {"02. Refinamento","06. Aprovação","07. Desenvolvimento","09. Deploy"})

// Backlog
Backlog = CALCULATE(COUNTROWS(Demandas), Demandas[fase]="01. Backlog")
4

Atualização Automática via Webhook

Configure um webhook no Kanboard (Configurações → Webhook) apontando para o endpoint de atualização do Power BI. Token do webhook: 5dbd35...

Para atualização programada no Power BI Service: acesse o dataset EBL Fast Track Salesforce → Atualização Agendada → Definir frequência diária.

CSV para Importação Manual

Arquivo CSV com todas as 105 demandas disponível para importação direta no Power BI Desktop:

Obter Dados → Texto/CSV → kanboard_dados_real.csv
Separador: ponto e vírgula (;) | Encoding: UTF-8 BOM

Campos disponíveis:
id | cherwell | rdm | titulo | fase | resp | area | tipo | pri | status | prazo | projeto | obs
